## UV practice

| Situation                      | Command                                        |
| ------------------------------ | ---------------------------------------------- |
| New project                    | `uv init`                                      |
| Project Include pyproject.toml | `uv sync`                                      |
| Only have requirements.txt     | `uv init --bare && uv add -r requirements.txt` |
| Nothing                        | `uv init --bare && uv add <packages>`          |
| Add dependencies               | `uv add <package>`                             |
| Add dev dependencies           | `uv add --dev <package>`                       |
| Production env                 | `uv sync --no-dev`                             |
| run project                    | `uv run src/myproject/main.py`                 |
